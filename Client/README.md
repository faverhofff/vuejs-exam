# examen-vue-app

## Project setup
```
npm install
```

### Run tests
```
npm testcafe -g 

cd server 

node index.js

cd client

npm run server

npm run test:all

note: you must have "Chrome" installed
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm build
```
